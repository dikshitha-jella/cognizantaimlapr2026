## Features

### Account Types
- **Savings Account**: Interest-bearing accounts with minimum balance requirements
- **Current Account**: Business accounts with overdraft facilities

### Customer Types
- **Individual**: Personal customer accounts with age tracking
- **Corporate**: Business customer accounts with company type

### Transaction Types
- **Direct Debit**: Scheduled payment transactions
- **External Transaction**: Transfers between branches

### Banking Operations
- Customer management (add, delete, edit)
- Account management (open, close, find)
- Transaction processing
- Banking group organization

## Commands

### Run Tests
```bash
source bankingenv/bin/activate
pytest tests/ -v
```

### Generate Test Reports
```bash
pytest tests/ --html=reports.html --self-contained-html -v --cov=src --cov-report=html
```

### Build Documentation
```bash
cd docs
make html
```

### Clean Generated Files
```bash
cd docs
make clean
```
